# Spyter
A speed typer, therefore Spyter

## Build and Run
### Build (Compile only)
```bash
mvn compile # -pl <module>)
```
### Build and Run Tests
```bash
mvn test # -pl <module>
```
### Build and Package (JAR)
```bash
mvn package # -pl <module>
```
### Run Main Class
```bash
mvn exec:java # -pl <module>
```
